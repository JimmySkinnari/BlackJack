# BlackJack


##Beskrivning


Uppgiften handlar om att skapa en webblösning där det går att spela ett kortspel. Spelet
måste visa ett antal spelkort i bilder på skärmen och det skall finnas en spelare som spelar
mot datorn. Använd bildfilerna på olika kort som finns bifogat. Krav för lösningen:

- Skall vara ett känt kortspel som Black Jack (enklaste spelet) eller någon form av Poker.

- Skall innehålla ett antal spelkort i form av bilder som visas på skärmen. Det skall tydligt
gå att se vad spelaren och datorn gör dvs korten som syns skall varieras beroende på
vilka drag som görs.

- Det skall tydligt gå att se när spelet är slut och det skall då gå att välja att spela en
gång till.

- Det är tillåtet att titta på kodexempel på nätet men inte att kopiera hela lösningar. Du
måste kunna förklara din kod och berätta vad varje del gör.

- Lösningen måste bygga på att jQuery används och du måste även använda SASS/LESS
för designen. Node.js skall användas för att hantera kod som går mot en databas. Så
man måste sätta upp en webbserver med routes som gör databasanrop.

- Det skall finnas en enkel databas som hanterar användare. För att man skall kunna
spela måste man logga in. Databasen skall även hålla reda hur mycket användaren har
i kassan. Det är pengar som man kan satsa när man spelar. Har spelaren inget i kassan
kan man inte spela utan måste fylla på. När en spelare spelar en omgång väljer man
hur mycket man satsar och kan då vinna pengar som fylls på i kassan. Om man förlorar
försvinner pengarna man satsat från kassan.

- Designen måste kännas genomarbetad och skall följa de riktlinjer som vi pratat om på
lektionerna. För koden gäller samma sak. Den skall fungera och skrivas
objektorienterat dvs. ni skall använda det vi har gått igenom på lektionerna kring
objektorientering. Lösningen skall gå att köra utan fel.
